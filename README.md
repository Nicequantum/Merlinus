## Key Features

- Voice-first input optimized for shop floor use
- Grok AI for high-quality warranty story generation
- AES-256-GCM encryption for sensitive data
- Tamper-evident hash-chained audit trail
- Professional branded PDF generation
- Stable, reliable UI designed for dealership environments

---

## How It Works

1. Technician logs in and opens a repair order
2. Dictates or enters repair details
3. Grok AI generates a professional warranty narrative
4. Every action is automatically logged in the secure audit trail
5. Technician reviews and exports a branded PDF

---

## Security & Compliance

- AES-256-GCM encryption of sensitive data
- Immutable SHA-256 hash-chained audit logging
- Signed Data Processing Agreement with xAI required before production use
- Session revocation on password change
- Private, session-protected image storage

---

## Common Failure Modes & Troubleshooting

| Issue | Symptom | Fix |
|-------|---------|-----|
| **Grok Timeout** | Long loading or timeout | Shorten input and retry |
| **Voice Input Issues** | Microphone not responding | Allow permission in Chrome/Edge |
| **PDF Generation Fails** | PDF download fails | Fill all required fields first |
